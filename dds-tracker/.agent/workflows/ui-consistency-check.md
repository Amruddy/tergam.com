# UI Consistency Check

Run this checklist before finishing UI work:

1. Changed components reuse `components/ui.tsx` primitives where possible.
2. New buttons use `Btn` or `IconBtn`.
3. New page headers use `PageHeader`.
4. New cards match existing card surfaces.
5. Mobile layout does not overflow.
6. Dark mode classes are present where light styles changed.
7. No new decorative visual system was introduced.
